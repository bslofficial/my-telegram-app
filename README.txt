BSL Candy Match - PRO Telegram Mini Game

Features:
- Telegram Mini App Ready
- Firebase Leaderboard & Withdraw Requests
- Rewarded Ads (Adsgram)
- Daily Reward
- Spin Wheel Bonus
- Invite & Earn (Referral)
- Anti-cheat basic check

Setup:
1. Replace Firebase config in firebase.js
2. Replace Adsgram blockId in game.js
3. Upload all files to GitHub
4. Enable GitHub Pages
5. Connect WebApp URL to your Telegram Bot via BotFather
